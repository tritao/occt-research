# Agent instructions (OCCT research)

## Hard rules
- Treat `./occt` as READ-ONLY. Never modify upstream OCCT files.
- Put research outputs in:
  - `./notes` (maps + dossiers)
  - `./tools` (scripts)
- Workflow scaffolding may be edited in:
  - `./prompts` (prompt templates)
  - `./backlog` (Backlog.md config/tasks)
- Avoid creating new files under `./oracles` (prefer `./repros/...`).
