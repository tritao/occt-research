# Backlog-driven prompts

Use these when working via Backlog.md tasks (with Backlog MCP tools).

- `prompts/backlog/task_generation.md`: seed lanes and create lane tasks.
- `prompts/backlog/map_task.md`: execute a `type:map` task.
- `prompts/backlog/dossier_task.md`: execute a `type:dossier` task.
- `prompts/backlog/repro_oracle_task.md`: execute a `type:repro` / `type:oracle` task.
- `prompts/backlog/work_task.md`: generic protocol for working a backlog task.
